# Startup Acquisition Analysis

## Overview
This project analyzes startup acquisition data to uncover patterns in funding, valuation, and acquisition trends. The goal is to identify factors that influence startup success and acquisition outcomes using data analysis techniques.

---

## Objectives
- Analyze startup funding and acquisition trends  
- Identify key factors leading to acquisitions  
- Study valuation patterns across startups  
- Extract insights for investors and founders  

---

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## Dataset
- Source: Startup Acquisition Dataset (Kaggle)  
- Contains information on:
  - Startup names  
  - Funding amounts  
  - Acquisition details  
  - Industry/domain  
  - Dates and financial metrics  

---

## Dataset Setup
To run this project:

1. Download the dataset from Kaggle  
2. Place the dataset file in your project folder  
3. Open `startup_acquired_project.ipynb` in Jupyter Notebook  
4. Run all cells step by step

## Key Analysis Performed

### Data Cleaning
- Handled missing values  
- Converted date formats  
- Cleaned inconsistent entries  

### Exploratory Data Analysis (EDA)
- Funding distribution  
- Acquisition trends over time  
- Industry-wise analysis  
- Top acquired startups  

### Visualization
- Bar charts  
- Line graphs  
- Distribution plots  
- Correlation analysis  

---

## Key Insights
- Startups with higher funding tend to have a higher probability of acquisition  
- Certain industries dominate acquisition activity  
- Acquisition trends vary across years  
- A small number of startups achieve very high valuations  

---

## Business Insights
- Investors should focus on high-growth sectors  
- Early funding plays an important role in acquisition success  
- Market trends influence acquisition timing  

---

## Limitations
- Limited dataset size  
- No real-time data  
- External factors such as market conditions are not included
